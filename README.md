# @oudom/create-react-tailwind

Create a React + TypeScript + Tailwind CSS project in one command.

## Website
https://crt.oudom.dev

## Usage
```bash
npx @oudom/create-react-tailwind my-app
cd my-app
npm run dev
```

## What's included

- React
- TypeScript
- Tailwind CSS v4
- Vite
